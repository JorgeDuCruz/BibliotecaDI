conexionBD

class Codigo.conexionBD.ConexionBD(rutaBd)[fuente]

Bases: object

Clase para gestionar la conexión y operaciones CRUD en una base de datos SQLite.

Esta clase está orientada a la gestión de una biblioteca, manejando las tablas de libros y sus autores, asegurando la integridad referencial mediante claves foráneas.

actualizaRexistro(updateSQL, *parametros)[fuente]

Actualiza un registro existente mediante su ID.

Parámetros:

  • updateSQL (str) – Sentencia UPDATE con marcadores “?”.

  • parametros – Nuevos valores incluyendo el ID al final para la cláusula WHERE.

borraRexistro(borraSQL, *parametros)[fuente]

Elimina un registro de la base de datos.

Si se borra un autor, se borrarán sus libros en cascada automáticamente gracias a la configuración de la clave foránea.

Parámetros:

  • borraSQL (str) – Sentencia DELETE con marcador para el ID.

  • parametros – ID del registro a eliminar.

conectaBD()[fuente]

Crea la conexión con la base de datos SQLite y habilita el soporte para claves foráneas.

Utiliza el comando PRAGMA para asegurar que las relaciones ON DELETE CASCADE funcionen correctamente en el motor SQLite.

consultaConParametros(consultaSQL, *parametros)[fuente]

Ejecuta una consulta SQL de selección utilizando parámetros de seguridad.

Parámetros:

  • consultaSQL (str) – Sentencia SQL con marcadores de posición “?”.

  • parametros – Valores para filtrar la consulta.

Devuelve:

Lista de tuplas con los resultados filtrados.

Tipo del valor devuelto:

list

Ejecuta una consulta SQL de selección sin parámetros.

Parámetros:

consultaSQL (str) – Sentencia SQL (SELECT) a ejecutar.

Devuelve:

Lista de tuplas con los registros resultantes.

Tipo del valor devuelto:

list

creaCursor()[fuente]

Crea el objeto cursor necesario para ejecutar sentencias SQL.

Requiere que la conexión haya sido establecida previamente mediante conectaBD().

crearTablas()[fuente]

Crea las tablas “autores” y “libros” estableciendo una relación 1:N.

Un autor puede tener muchos libros, pero un libro pertenece a un solo autor. La tabla libros incluye restricciones de validación para la puntuación.

engadeRexistro(insertSQL, *parametros)[fuente]

Inserta un nuevo registro (Libro o Autor) en la base de datos.

Parámetros:

  • insertSQL (str) – Sentencia INSERT con marcadores “?”.

  • parametros – Datos del registro a insertar.

pechaBD()[fuente]

Realiza el cierre seguro del cursor y de la conexión con la base de datos.

Bases: object

Clase principal que orquestra el inicio de la aplicación de biblioteca.

Esta clase se encarga de establecer la conexión inicial con la base de datos, asegurar que la estructura de tablas exista y lanzar la primera ventana de la interfaz de usuario (VentanaSeleccion).

Este método mantiene la aplicación en ejecución y a la espera de interacciones por parte del usuario hasta que se cierre la ventana principal.

class Codigo.ventana_seleccion.VentanaSeleccion(*args: Any, **kwargs: Any)[fuente]

Bases: Window

Ventana de menú principal que sirve como orquestador de la aplicación.

Esta clase permite al usuario navegar entre la gestión de autores y la gestión de libros. Utiliza un sistema de ventanas persistentes para evitar la recreación constante de objetos y mejorar el rendimiento.

abrir(ventana)[fuente]

Gestiona la transición entre el menú principal y las ventanas secundarias.

Oculta la ventana de selección, refresca los datos de la base de datos en la ventana destino y la presenta al usuario.

Parámetros:

ventana (Gtk.Window) – Instancia de la ventana que se desea mostrar.
